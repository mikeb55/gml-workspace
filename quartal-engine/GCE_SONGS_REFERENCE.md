## Chord-to-Scale Mapping Guide for Quartal Engine

### For Dominant 7th chords (G7, C9, D7#9, etc.):
- Use **Mixolydian** mode
- Example: G7 → G mixolydian, C9 → C mixolydian

### For Minor chords (A-7, D-7, etc.):
- Use **Dorian** mode  
- Example: A-7 → A dorian, D-7 → D dorian

### For Diminished chords (G#o7, C#o7, etc.):
- Use **Locrian** mode
- Example: G#o7 → G# locrian, C#o7 → C# locrian

### For Major 7th chords (C^7, F^7#11, etc.):
- Use **Lydian** or **Mixolydian** mode
- Example: C^7 → C lydian or C mixolydian

### For Half-diminished (A-7b5, D-7b5, etc.):
- Use **Locrian** mode
- Example: A-7b5 → A locrian
