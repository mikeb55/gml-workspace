# Note Duration Commands - Complete Examples

## How to Specify Note Duration

Add the duration keyword to your command:

### Supported Durations

- **Half notes**: `half`, `half notes`
- **Quarter notes**: `quarter`, `quarter notes` (default)
- **Eighth notes**: `eighth`, `eighth notes`, `8th`, `8th notes`
- **Sixteenth notes**: `sixteenth`, `16th`, `16th notes`

## Sample Sentences for 3 Different Harmonized Progressions

### Example 1: Slow, Sustained Progression (Half Notes)
```
Generate C major quartals, 4 bars, half notes
```
**Result:** 4 bars, 12 notes (3-note quartals), half-note duration
**Use case:** Slow, sustained harmonic progression with long, ringing chords

### Example 2: Medium Tempo (Quarter Notes)
```
Generate D dorian quartals, 4 bars, quarter notes
```
**Result:** 4 bars, 12 notes (3-note quartals), quarter-note duration
**Use case:** Standard tempo, common time feel, typical harmonic rhythm

### Example 3: Fast, Rhythmic (Eighth Notes)
```
Generate E minor quartals, 8 bars, eighth notes
```
**Result:** 8 bars, 24 notes (3-note quartals), eighth-note duration
**Use case:** Fast, rhythmic progression with quick chord changes

## Command Format

```
Generate [ROOT] [SCALE] quartals, [N] bars, [DURATION]
```

Where `[DURATION]` can be:
- `half notes` or `half`
- `quarter notes` or `quarter` (default if omitted)
- `eighth notes`, `eighth`, `8th notes`, or `8th`
- `sixteenth notes`, `sixteenth`, `16th notes`, or `16th`

## Duration Comparison

| Duration | Beats per Bar (4/4) | Use Case |
|----------|---------------------|----------|
| Half notes | 2 notes per bar | Slow, sustained |
| Quarter notes | 4 notes per bar | Standard tempo |
| Eighth notes | 8 notes per bar | Fast, rhythmic |
| Sixteenth notes | 16 notes per bar | Very fast, dense |

## Tips

1. **Half notes**: Best for ballads, slow progressions, ambient textures
2. **Quarter notes**: Most versatile, works for most styles
3. **Eighth notes**: Good for jazz, funk, faster progressions
4. **Sixteenth notes**: Very fast, use sparingly for dense textures
